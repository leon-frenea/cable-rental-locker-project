## Projet d'Electronique en groupe.

La consigne était de design un casier d'emprun de cables (Banane, BNC et sonde) avec :
- Un lecteur RFID
- Un servo moteur verouilleur de porte et un bouton fin de course
- Une base de donnée de l'historique des emprunts
- Un système d'antivol par acceleromètre avec une réaction sonore et une alerte par mail en réponse

Toute la documentation du projet se trouve en commentaire dans les divers fichiers de code
